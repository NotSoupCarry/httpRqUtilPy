# Setup
To run this tool, you need to create a configuration file to store the variables.

## Configuration
Create a file named `config.py` and add the following variables:

###  Env Variables
* **BASE_URL**: The base URL of the server.
* **RQ_PATH**: The specific endpoint path.
* **JWT_TOKEN**: Your Bearer authentication token.

* **NUM_REQUESTS**: Total number of requests to perform.
* **MAX_WORKERS**: Number of concurrent threads to utilize.
